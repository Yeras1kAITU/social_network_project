API Documentation

Complete reference for StudyConnect Posts API. Create, read, update, and delete study posts.

Database Information

Database

MongoDB (studyconnect database)

Collection

posts

Auto-initialized

On server startup

Document Schema

{
    _id: ObjectId,
    postId: Number,
    title: String,
    content: String,
    author: String,
    category: String,
    created_at: Date,
    updated_at: Date,
    likes: Number,
    is_published: Boolean
}

API Endpoints

GET /api/posts

Retrieve all posts. Supports query parameters for filtering.

Test Endpoint
POST /api/posts

Create a new post. Requires authentication.

Body: {"title": "string", "content": "string", "author": "string", "category": "string"}

PUT /api/posts/:id

Update an existing post. Requires authentication.

DELETE /api/posts/:id

Delete a post (soft delete). Requires authentication.

API Examples with curl 

# Get all posts
curl http://localhost:3000/api/posts

# Get post with ID 1
curl http://localhost:3000/api/posts/1

# Get categories
curl http://localhost:3000/api/posts/categories

# Get statistics
curl http://localhost:3000/api/posts/stats

# Health check
curl http://localhost:3000/api/health
